# 🪓 Hangman Game 

A classic word-guessing game implemented in Python. Players attempt to save the "hangman" by guessing a hidden word one letter at a time within a limited number of attempts.

## 🚀 Features
- **Random Word Selection**: Pulls words from a predefined list or external text file.
- **ASCII Art**: Visual representation of the hangman that updates with every wrong guess.
- **Live Updates**: Displays correctly guessed letters in their correct positions.
- **Input Validation**: Detects repeated guesses and invalid characters.
- **Win/Loss Conditions**: Clear messaging when you save the man or run out of lives.
